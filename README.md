**Draedon** is a modular, cross-platform reconnaissance framework built for passive and informational intelligence gathering. Designed around a clean plugin architecture ‚ÄĒ drop a file into `modules/`, and it works.

---

## Modules

| Name | Description | Target |
|------|-------------|--------|
| `geoip` | IP geolocation via ip-api.com | `ip` |
| `whois` | Domain registration info | `domain` |
| `dns` | DNS records ‚ÄĒ A, MX, TXT, NS, CNAME, SPF, DMARC | `domain` |

---

## Installation

```bash
git clone https://github.com/yourusername/draedon.git
cd draedon
pip install python-whois dnspython
```

---

## Usage

```bash
python main.py 8.8.8.8
python main.py google.com --type domain
python main.py --list
```
